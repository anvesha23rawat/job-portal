# Job Portal App - Frontend

A modern, responsive, and full-featured frontend for a Full-Stack Job Portal application built using the MERN stack. This interface allows students to search and apply for jobs while offering recruiters a streamlined portal to manage job postings.

##  Features Implemented
* **User Authentication:** Clean and interactive Signup and Login forms with specific role selection (Student / Recruiter).
* **Dynamic Navigation:** Responsive Navbar featuring active states, profile picture tracking, and a smooth user profile dropdown action menu.
* **Modern UI:** Crafted with highly polished, responsive layouts using Tailwind CSS.
* **State Management:** Integrated state tracking to maintain global user sessions across individual pages.

##  Tech Stack Used
* **Frontend Library:** React.js (via Vite)
* **Styling:** Tailwind CSS
* **Icons & Components:** Lucide React / Radix UI elements
